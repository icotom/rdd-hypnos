# rdd-hypnos
Fiche de personnage pour Reve de Dragon sur Roll20, sur la base de la fiche creee par Phoxounet (disponible sur la base de donnees de feuilles de personnage de Roll20).

Principales differences:
- pre-page de selection du type de feuille (feuille de personnage joueur ou feuille simplifiee pour les PNJs)
- les jets de competences montrent le nombre de points de taches obtenu
- les jets de des ont un style different en fonction du type de jets (jet de caracteristiques purs; jets de competences; jets de Reve)
- choix de l'Heure de naissance en menu dropdown
- toggle Haut Revant / Vrai revant (affichage des sorts et des voies draconiques pour les HR; affichage d'une section Vrai Reve pour les VR)
- table de resolution deplacee en bas de la page de personnage
- simplification de la partie competence
- ajout d'un boutton pour le jet de Stress
- ajout de feuilles de personnage simplifiees pour les PNJs, creatures, et entite de cauchemar (leurs jets ont un style different des jets de des PJs)
